# Notifications Page

## Purpose
This screen shows notifications such as:
- New appointments
- Rescheduled appointments
- Cancelled appointments

## Interactions
- Clicking different tabs (All / Scheduled / Rescheduled / Cancelled) changes what is displayed.
- Clicking an item marks it as read and opens its details.

## State Changes
- activeTab: which tab is selected
- notifications: list of notifications
- selectedNotification: currently opened notification
