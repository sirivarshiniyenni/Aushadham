# Appointments Page

## Purpose
Displays today's appointments with patient details, timing, test type, and a "Join Now" button.

## Interactions
- User changes filter (All / In-clinic / Tele)
- Clicking "Join Now" starts video consultation
- Clicking an appointment row opens full details

## State Changes
- filter: the active filter
- appointments: list of appointments
- activeAppointment: the selected appointment
